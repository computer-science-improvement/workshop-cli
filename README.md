## Usage

1. git clone
2. npm install
3. create API_KEY - [here](https://www.omdbapi.com/)
4. put api key in .env

## Requirements:
 #### node > v19.7.0 

## Run

1. node index.js --search <(MOVIE, SERIES, EPISODE) **NAME**>
2. Enjoy
